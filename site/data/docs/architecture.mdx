# Architecture by runtime

Node and Orez Lite have separate architectures. Choose the runtime whose machinery you want to understand:

<CardGrid>
  <DocCard href="/docs/node/architecture" title="Node architecture">
    Stock zero-cache coordinated with PGlite or real Embedded Postgres, plus replica
    storage and lifecycle services.
  </DocCard>
  <DocCard
    href="/docs/orez-lite/architecture"
    title="Orez Lite architecture"
    badge="Preview"
  >
    A deterministic Rust core with native and WASM hosts, per-namespace SQLite, HTTP
    pull/push, and wake.
  </DocCard>
</CardGrid>
