# Configuration by runtime

Configuration is runtime-specific:

- [Node configuration and CLI](/docs/node/configuration) covers the `orez` command, `orez.config.ts`, environment variables, ports, storage, hooks, and zero-cache options.
- [Node programmatic API](/docs/node/api) covers `startZeroLite()`, Vite integration, lifecycle controls, and local S3.
- [Orez Lite setup](/docs/orez-lite/cloudflare) covers the application endpoints,
  `on-zero` client transport, and Cloudflare host boundary.
