# The sync engine is now Orez Lite

The Rust and SQLite implementation of Zero's server side is now **Orez Lite**.
Its documentation has moved to the [Orez Lite overview](/docs/orez-lite).

The internal Rust crate names currently remain `sync-core`, `sync-native`, and `sync-wasm`. Orez Lite is the public product name across the documentation.
