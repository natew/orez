# Testing by runtime

Orez crosses PostgreSQL, SQLite, WebAssembly, and Zero protocol surfaces. Each runtime documents evidence next to the contract it exercises, so a passing test has a clear scope.

## Reading an evidence claim

For each claim, the runtime page identifies the layer that ran, the implementation boundaries it crossed, and the asserted result. Compiler fixtures establish translation behavior. Process scenarios establish the behavior of a running stack under the stated setup.
